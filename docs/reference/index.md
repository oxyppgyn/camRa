# Package index

## MegaDetector/SpeciesNet JSON Data

Functions for utilizing JSON files containing MegaDetector and
SpeciesNet detections and classifications.

- [`megadet_filter()`](https://oxyppgyn.github.io/camRa/reference/megadet_filter.md)
  : Filter MegaDetector/SpeciesNet JSON Files by Image
- [`megadet_flatten()`](https://oxyppgyn.github.io/camRa/reference/megadet_flatten.md)
  : Flatten MegaDetector/SpeciesNet JSON Detection Data
- [`megadet_get_counts()`](https://oxyppgyn.github.io/camRa/reference/megadet_get_counts.md)
  : Count Number of Instances of MegaDetector/SpeciesNet Categories in
  Images
- [`megadet_get_info()`](https://oxyppgyn.github.io/camRa/reference/megadet_get_info.md)
  : Get Values from MegaDetector/SpeciesNet JSON Files
- [`specnet_reclassify()`](https://oxyppgyn.github.io/camRa/reference/specnet_reclassify.md)
  : Update Classification Information in SpeciesNet JSON Data

## Image Manipulation

Functions for manipulating and extracting information from images.

- [`img_apply()`](https://oxyppgyn.github.io/camRa/reference/img_apply.md)
  : Apply a Summary function to an Image
- [`img_difference()`](https://oxyppgyn.github.io/camRa/reference/img_difference.md)
  : Calculate Differences Between Two Images
- [`img_extract_text()`](https://oxyppgyn.github.io/camRa/reference/img_extract_text.md)
  : Extract Text from Image Using OCR
- [`img_luminosity()`](https://oxyppgyn.github.io/camRa/reference/img_luminosity.md)
  : Calculate Luminosity/Brightness of an Image
- [`convert_bbox()`](https://oxyppgyn.github.io/camRa/reference/convert_bbox.md)
  : Convert between BBox Formats

## LILA

Functions for interacting with the Living Library of Alexandria (LILA).

- [`LILA_download()`](https://oxyppgyn.github.io/camRa/reference/LILA_download.md)
  : Download LILA Image Files
- [`LILA_list_datasets()`](https://oxyppgyn.github.io/camRa/reference/LILA_list_datasets.md)
  : Get a List of Datasets from LILA BC
- [`LILA_list_files()`](https://oxyppgyn.github.io/camRa/reference/LILA_list_files.md)
  : List Files in LILA Datasets

## Timelapse

Functions for interacting with Timelapse .ddb and .tdb files.

- [`tl_ddb_extract()`](https://oxyppgyn.github.io/camRa/reference/tl_ddb_extract.md)
  : Get Data Table from Timelapse Database
- [`tl_ddb_insert()`](https://oxyppgyn.github.io/camRa/reference/tl_ddb_insert.md)
  : Insert New Values for a Timelapse Database Column
